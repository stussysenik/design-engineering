# Portfolio Steering Pack

This folder is the operating system for the next version of the public `stussysenik` profile.

As of April 1, 2026, the goal is not to look broader.
The goal is to look sharper.

## Core Thesis

The strongest public story is:

`I build AI-native tools with excellent interaction design across iOS, web, and CLI.`

That is the center.
Graphics, CV, shaders, WebGPU, and systems knowledge are supporting depth, not the headline.

## Files

- `PRESENT.md`
  Current state, portfolio diagnosis, GitHub cleanup plan, repo rename guidance, and Windmill fit assessment.
- `VISION.md`
  Long-term direction, what to double down on, what deeper roles would require, and the next flagship projects.

## Immediate Use

1. Read `PRESENT.md` and apply the GitHub cleanup plan.
2. Read `VISION.md` and choose the primary lane.
3. Rewrite the GitHub profile README around one sentence and 4 pinned projects.
4. Stop adding net-new public repos until the flagship set is coherent.

## Working Rule

For the next phase:

- fewer repos
- clearer names
- better READMEs
- more proof
- more essays tied to shipped artifacts
